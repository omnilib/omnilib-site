Projects
========

Omnilib is a group of high quality, MIT licensed software libraries developed
under a common, inclusive :ref:`code-of-conduct`. Omnilib is committed to
providing a welcoming and open space for all contributors who adhere to these
rules.


:ref:`aioitertools <project-aioitertools>`
------------------------------------------

itertools and more for AsyncIO.


:ref:`aiomultiprocess <project-aiomultiprocess>`
------------------------------------------------

The best of AsyncIO and multiprocessing combined.


:ref:`aiosqlite <project-aiosqlite>`
------------------------------------

Friendly, async interface to sqlite databases.


:ref:`aql <project-aql>`
------------------------

Async, typing-friendly query generator for SQL databases.


:ref:`attribution <project-attribution>`
----------------------------------------

Changelog generator and release tag manager.


:ref:`stdlibs <project-stdlibs>`
--------------------------------

Simple list of top-level packages in Python's stdlib.


:ref:`thx <project-thx>`
------------------------

Rapid development assistant, and fast command runner for Python projects.


:ref:`trailrunner <project-trailrunner>`
----------------------------------------

Simple library for walking paths on the filesystem and running functions.


:ref:`µfmt <project-ufmt>`
--------------------------

Safe, atomic code formatter for Python built on top of
`black <https://black.rtfd.io>`_ and `µsort <https://usort.rtfd.io>`_.
